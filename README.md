# COVID-19 Detection System

## 📌 Project Description

A deep learning-based system that detects COVID-19 from chest X-ray images using Convolutional Neural Networks (CNN) and provides real-time predictions through a web interface.

## 🚀 Features

* Image-based COVID-19 detection
* Trained CNN model using PyTorch
* Real-time prediction using Flask web app
* 92% validation accuracy
* User-friendly interface

## 🛠 Tech Stack

* Python
* PyTorch
* Flask
* HTML, CSS

## ▶️ How to Run

1. Clone the repository
2. Install dependencies

   ```
   pip install -r requirements.txt
   ```
3. Run the application

   ```
   python app.py
   ```
4. Open in browser

   ```
   http://127.0.0.1:5000/
   ```
